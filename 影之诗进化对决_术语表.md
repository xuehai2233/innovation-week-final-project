# 影之诗：进化对决 规则术语表

资料来源：

- 中文规则书：《影之诗：进化对决》综合规则 ver. 1.7.0，最终更新日 2024-06-13。
- 日文规则书：Shadowverse EVOLVE 総合ルール ver. 1.26.0，最终更新日 2026-05-18。

说明：

- “简中译名”优先沿用中文规则书和既有简中卡牌文本用语。
- 标注“暂译”的条目，是日文 1.26.0 中存在、但中文 1.7.0 规则书未覆盖或未能确认官方简中译名的术语。
- 规则号以日文 1.26.0 为主；若中文规则书已有对应概念，译名按中文规则书体系处理。

## 推荐统一写法

| 推荐写法 | 不建议混用 | 备注 |
| --- | --- | --- |
| 主战者 | 领导者、领袖 | 中文规则书使用“主战者”。 |
| 随从 | 侍从、Follower | 中文规则书使用“随从”。 |
| 护符 | 咏唱物、护身符 | 中文规则书使用“护符”。 |
| 法术 | 魔法、Spell | 中文规则书使用“法术”。 |
| 费用 | 消费 | 规则书偏“费用”；卡牌效果中“消费 X”也常见，可按语境使用。 |
| 生命值 | 体力 | 中文规则书将主战者/随从的体力译作“生命值”。 |
| 横置 / 竖置 | 横放 / 重置 | 中文规则书使用“横置/竖置”。 |
| 置于墓场 | 送墓、进墓 | 规则文本建议保持“置于墓场”。 |
| 消失 | 除外、放逐 | 既有简中卡牌文本使用“消失”。 |
| 检查时点 | 检查时间点 | 中文规则书使用“检查时点”。 |
